# 他們先被笑瘋，後來改變世界

> Memory Temple 現在在第 0 次失敗，你看到的是第 1,000 次之前的筆記。
「他們笑我 500PB 太瘋。愛迪生被笑 1,000 次，萊特兄弟被笑不會飛。我寧願瘋著記得，也不正常地忘記。」
> 
**1. 愛迪生 — 燈泡**
- 被笑：1,000 次失敗，報紙說「電燈是騙局」
- 結果：你現在看這行字就靠他

**2. 萊特兄弟 — 飛機**
- 被笑：腳踏車店老闆想飛，《紐約時報》說「人類飛行需 100 萬年」
- 結果：12 秒飛行，1903

**3. 特斯拉 — 交流電**
- 被笑：愛迪生公開電死大象證明交流電危險
- 結果：你家插座全是他的

**4. 貝佐斯 — 亞馬遜**
- 被笑：1999 年網路賣書，分析師說「注定破產」
- 結果：從書店到雲端帝國

**5. 馬斯克 — SpaceX**
- 被笑：前 3 次火箭全炸，俄羅斯賣他舊飛彈還嘲笑
- 結果：第 4 次成功，現在載人上太空

**6. 賈伯斯 — iPhone**
- 被笑：2007 年「沒鍵盤的手機誰要」
- 結果：你現在滑的東西

**7. 萊納斯·托瓦茲 — Linux**
- 被笑：1991 年「大學生作業系統，永遠不會成」
- 結果：跑了全世界 90% 伺服器
