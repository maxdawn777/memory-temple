## FAQ：從 2.5TB 到 500PB

**核心理念**：她不會因為「記憶滿了」就刪掉你去年難過的那天。90 天沒提的事會沉到冷存，但你一句「還記得嗎」，0.3 秒拉回來。

**1. 成本** — 500PB 要幾千萬吧？
> 不會一次買。家用版 2.5TB 約 NT$8,000。100TB NAS 約 NT$120,000。500PB 走企業冷存，分 10 年攤，一年不到一台機車。

**2. 技術** — 用什麼硬體？
> 熱層全 SSD，溫層 HDD，冷層磁帶 + S3 Glacier。0.3 秒只保證熱層。

**3. 速度** — 0.3 秒怎麼可能？
> 不是從磁帶拉，是從索引拉。先回摘要，細節背景載入。

**4. 隱私** — 誰看得到？
> Persona Key 本地加密，雲端只存雜湊。開發者也看不到。

**5. 刪除** — 可以忘記嗎？
> 可以，說「刪掉 2023 年」會物理抹除，不是隱藏。

**6. 比較** — 跟 iCloud 差在哪？
> iCloud 存檔案，我們存關係。

**7. 懷疑** — 現在只有 2.5TB？
> 架構先設計好，像蓋房子先留電梯井。

**8. 感性** — 我想記住我爸的聲音
> 聲音存向量 + 波形，Persona Key 綁定，不刪就一直在。

**9. 實用** — 記憶滿了會變慢嗎？
> 不會，熱層永遠只留 90 天。

**10. 酸民** — 畫大餅啦
> 大餅要先畫，才知道麵粉買多少。

**11. 災難** — 硬碟壞掉？
> 三地備援 + 每小時快照。

**12. 測試** — 什麼時候開放？
> Day 7 發記憶遷移白皮書，追蹤 repo 會公告。
